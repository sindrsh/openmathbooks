\subsection*{Oppgave 1}
\alg{
\text{Pris for voksen} = x && \text{Pris for barn} = y
}
\begin{align}
	2x + y = 400 \label{opg1a}\\
	x + y = 260 \label{opg1b}
\end{align}
Vi trekker \eqref{opg1b} i fra \eqref{opg1a}:
\alg{
2x+y-(x+y) &= 400 -260\\
x &= 140
}
En barnebillett koster 140 kroner.
